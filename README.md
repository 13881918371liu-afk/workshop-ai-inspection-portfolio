# Workshop AI Inspection Portfolio

This repository hosts a static project portfolio for a workshop AI inspection and production-record prototype.

The site is intended for external review and contains only sanitized concept materials:

- `index.html`: portfolio homepage
- `architecture.html`: overall architecture
- `interaction.html`: employee-side and manager-side interaction design
- `sample-analysis.md`: sample form analysis and implementation notes

No company source code, credentials, private chat records, or raw production data are included.
